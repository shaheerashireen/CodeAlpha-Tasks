# E-Commerce Sales & Profitability Dashboard

This project analyzes e-commerce sales data using Tableau to identify trends, profitability, and regional performance.

# Objective
- Visualize sales and profit data for business insights
- Support data-driven decision-making

# Tool Used
- Tableau

# Key Insights
- Sales show an overall upward trend
- Technology category is the most profitable
- Some sub-categories generate losses despite high sales
- Consumer segment contributes the highest sales
